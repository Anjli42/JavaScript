1. What is Javascript

  • JavaScript is a lightweight, single-threaded, and interpreted compiled programming language.

  • JavaScript can be used for Client-side (Front-end development) developments as well as Server-side (Back-end development) developments.

  • dynamically typed language (Developers can use JavaScript to make websites dynamic and interactive.)

  • Handle HTML and CSS

  • Event Based Programming Language

  • Get and set cookies, ask questions to the visitor, show messages.

  • rules

    • start with => _, alpha, $
    • Assignment operator => ' = '

  • Types : 3

    • var: global scope var a = 10; // Initialization and decalration var a = "krish"; // Re-decalreation is possible a = "Jitu Bhaiya"; // Re-assignement is possible console.log(a);

    let: block scope let marks = 85; // Initialization and decalration // let marks = 90; // Re-decalreation is not possible marks = 95; // Re-assignement is possible console.log(marks);

    const: block scope const fav_Actress = "Madhubala"; // Initialization and decalration // const fav_Actress = "Alia";// Re-decalreation is not possible // fav_Actress = "Katrina"; // Re-assignement is not possible console.log(fav_Actress);

12. Function in JS

  • defined with the function keyword, followed by a name, followed by parentheses ()

  • Function names can contain letters, digits, underscores, and dollar signs (same rules as variables).

  • The parentheses may include parameter names separated by commas: (parameter1, parameter2, ...)

  • function is placed inside curly brackets: {}
